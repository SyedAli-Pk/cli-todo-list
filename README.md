# CLI To-Do List

A simple, command-line to-do list application written in C++.  
Keep track of your tasks easily, right from your terminal!

## Features
- Add new tasks  
- View all tasks  
- Mark tasks as done  
- Delete tasks  
- Simple and lightweight, no external dependencies  
- No file saving (tasks are lost after program exits)
